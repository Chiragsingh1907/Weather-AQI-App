## 📌 Project Description

This is a Python-based desktop application that shows real-time weather and air quality (AQI) data for Indian cities.

The system also predicts future temperature and AQI using a simple Machine Learning model.

## ⚙️ Features

* 📈 24-hour temperature prediction
* 🌫️ AQI monitoring and forecasting
* 🌡️ Heatmap of Indian cities
* 🏙️ City comparison
* 📊 Analytics dashboard

## 🧠 Concept Used

The project uses a sinusoidal model to represent daily patterns in temperature and AQI.

## 🛠️ Technologies Used

* Python
* NumPy, Pandas
* Matplotlib
* SciPy
* Scikit-learn
* CustomTkinter
* Open-Meteo API

## ▶️ How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the program:
   python weather_aqi_app.py

## 🌐 Data Source

* Open-Meteo API (real-time data)
